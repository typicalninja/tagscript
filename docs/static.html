Static Data (properties)

  • In basic form static Data is a object
  • It contains all the data regarding a script expressions all grouped and given correct types
  • Static Generator does NOT Care about the context (ex: variable can be undefined but will still be detected as a variable) , it will just output the parse data

Optimizing / Providing preGenStaticData

ctx.parse(script, preGenStaticData) / parse(.., ..., ..., preGenStaticData) both functions give you a chance
You can use this to optimize the parser as the it will create a new static Data every time a new script gets parsed if preGenStaticData is not present

Creating a new Static Data might take few milliseconds and can be optimized if we provided a cached static object to the parser

  • We can use a Database to store our static Data Object and provided it as preGenStaticData when the respective script needs to be run
